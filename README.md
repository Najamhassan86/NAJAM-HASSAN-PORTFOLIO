# Muhammad Najamuddin Hassan Portfolio

Static portfolio website for Muhammad Najamuddin Hassan, inspired by the visual style of SovaVis and populated from the provided CV.

## Stack
- HTML
- CSS
- Vanilla JavaScript

## Main Files
- `index.html` - site structure and content
- `style.css` - visual design and responsive layout
- `script.js` - navigation and reveal interactions

## Included Content
- Hero and personal intro
- About and professional focus
- Experience (SovaVis + Freelance)
- Project portfolio
- Technical and professional skills
- Education timeline
- Achievements, training, and activities
- Contact links and CV download

## Run
Open `index.html` directly in a browser.

## Notes
- CV files are available in the root directory.
- Profile image is loaded from `assets/images/profile.png`.
